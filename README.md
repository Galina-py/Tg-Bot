# Telegram Bot "Помощник"

Многофункциональный Telegram-бот для напоминаний о здоровье и расписании. 
Бот отправляет уведомления о необходимости пить воду, делать зарядку, 
а также предоставляет интересные факты о воде и расписание уроков.

## Функциональность

### Основные команды
- /start - запуск бота и приветственное сообщение
- /fact - получение случайного факта о воде

### Автоматические напоминания
- Питьевой режим: напоминания выпить стакан воды в 09:00, 14:00 и 18:00
- Зарядка: ежедневное напоминание в 06:30
- Факты о воде: отправка интересных фактов каждый день в 12:00
- Расписание уроков: напоминание о занятиях на следующий день (по будням в 15:00)

## Требования

- Python 3.7+
- Библиотеки:
  - pyTelegramBotAPI
  - schedule
  - datetime
  - threading
  - random

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Galina-py/Tg-Bot.git
   cd Tg-Bot
2. Установите зависимости:
  
   pip install pyTelegramBotAPI schedule
   
3. Настройте переменные окружения или отредактируйте файл:
   · TOKEN - токен вашего Telegram-бота (получить у @BotFather)
   · CHANNEL_ID - ID канала или чата для отправки сообщений
4. Запустите бота:
   ```bash
   python bot.py
